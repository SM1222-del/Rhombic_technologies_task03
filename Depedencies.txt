###Dependencies:

requests: For making HTTP requests to APIs.

folium: For creating and displaying interactive maps.

speech_recognition: For speech recognition (converting speech to text).

pyttsx3: For text-to-speech conversion.

webbrowser: For opening the map in a web browser (a built-in Python module). 

os: For file path manipulation (a built-in Python module).

###Installation Commands:
Install the required packages using pip. Run the following commands in your terminal or command prompt:

'pip install requests folium SpeechRecognition pyttsx3'
